🤖🎙️ Reachy Mini — Live Video Chat

Full-duplex · bilingual (中文/English) · ≤1.5s · local on Apple Silicon

A full-duplex, low-latency voice + video conversational app for Reachy Mini. It runs the VAD → ASR → LLM → TTS pipeline locally on your Mac (MLX), reacts with expressive motion, turns toward the speaker via DOA, and keeps vision-token usage tiny.

Install & run

./scripts/setup_mac.sh
source .venv/bin/activate
reachy-mini-live-chat --sim --stub   # hardware-free demo
reachy-mini-live-chat                # on the robot (daemon running)

This is a Reachy Mini Python app: install it and launch it from the robot dashboard or the CLI. Web control UI runs on :8042.
