Conversational Embodied Robot · Reference Architecture

通用具身对话机器人参考架构 — 实时语音大模型 × 工具调用 × 分层运动控制，适用于任意机器人形态

实时语音链路

  • • 全双工 WebSocket，16 kHz PCM 流式收发
  • • 服务端 VAD，支持随时打断（barge-in）
  • • 流式转写与语音回复同步下发
  • • 视觉帧以图片消息注入同一会话上下文

工具驱动行为

  • • 全部机器人行为由 LLM function-calling 触发
  • • 异步编排器后台并发执行、结果批量回传
  • • 动作 / 视觉 / 记忆 / 休眠 + MCP 远程工具
  • • 插件式接口，新增能力无需修改核心

分层运动控制

  • • 60 Hz 控制循环，每 tick 单一控制点下发姿态
  • • 主动作队列顺序执行（舞蹈 / 情绪 / goto）
  • • 叠加混合：呼吸、说话摆动、注视跟踪
  • • 硬件抽象层隔离具体机器人形态
